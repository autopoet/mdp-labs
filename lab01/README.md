# Lab01：Huffman 编码实验

本实验实现了 Huffman 编码与解码的基本流程，示例输入字符串为：

```text
helloworld
```

## 文件说明

- `huffman.js`：Huffman 编码、解码与验证代码。

## 运行方式

在 `lab01` 目录下执行：

```bash
node huffman.js
```

## 实验流程

1. 统计输入字符串中每个字符出现的频率。
2. 按频率从小到大排序，构造初始节点队列。
3. 每次取出频率最小的两个节点，合并成一个新节点。
4. 重复合并，直到只剩一个根节点，得到 Huffman 树。
5. 从根节点开始遍历：
   - 向左走记为 `0`；
   - 向右走记为 `1`。
6. 得到每个字符对应的 Huffman 编码字典。
7. 使用编码字典把原字符串编码为二进制串。
8. 根据 Huffman 树对二进制串进行解码。
9. 比较解码结果和原字符串，验证编码解码是否正确。

## 原理说明

Huffman 编码是一种可变长编码方法。出现频率高的字符使用较短编码，出现频率低的字符使用较长编码，从而减少整体编码长度。

Huffman 编码具有前缀码性质：任意一个字符的编码都不是另一个字符编码的前缀，因此解码时可以从左到右唯一确定每个字符。

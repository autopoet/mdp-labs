## 实验目标

PCA，Principal Component Analysis，中文叫主成分分析。

它的目标是：把高维数据投影到少数几个最重要的方向上，在降低维度的同时尽量保留原始数据中的主要信息。

本实验中，每个样本原来有 32 维颜色直方图特征。题目要求把它分别降到 1-31 维，并观察：

1. 原始数据总方差是多少；
2. 降到不同维度后还能保留多少方差；
3. 降维再重构后会产生多少均方误差。

## 原理讲解

### 1. 为什么要中心化

PCA 关注的是样本相对于整体平均水平的变化方向，而不是原始坐标位置。

所以第一步要对每一维特征求平均值，然后每个样本都减去这个平均值：

```text
Xc = X - mean(X)
```

其中：

- `X` 是原始特征矩阵；
- `mean(X)` 是每一维特征的均值；
- `Xc` 是中心化后的数据。

中心化之后，每一维特征的均值接近 0，这样后面计算协方差矩阵才符合 PCA 的基本假设。

### 2. 协方差矩阵表示什么

中心化以后，计算协方差矩阵：

```text
C = Xc^T Xc / N
```

其中：

- `N` 是样本数量；
- `C` 是一个 32 × 32 的矩阵；
- `C[i, i]` 表示第 i 维特征自己的方差；
- `C[i, j]` 表示第 i 维和第 j 维特征之间的相关程度。

协方差矩阵可以理解为：它描述了数据在不同方向上的分散程度。

### 3. 主成分是什么

PCA 要找的是一组新的坐标轴。

这些新坐标轴有两个特点：

1. 彼此正交，也就是互不重复；
2. 按照能解释的方差从大到小排序。

协方差矩阵的特征向量就是这些新方向，特征值就是数据在对应方向上的方差。

因此：

```text
特征值越大，说明这个主成分方向保留的信息越多。
```

### 4. 降维方差怎么计算

如果把数据降到 `k` 维，就保留前 `k` 个最大的特征值。

投影方差就是这些特征值之和：

```text
k 维投影方差 = lambda_1 + lambda_2 + ... + lambda_k
```

原始总方差是所有特征值之和：

```text
原始总方差 = lambda_1 + lambda_2 + ... + lambda_d
```

其中 `d = 32`。

### 5. 重构误差怎么理解

降维时，前 `k` 个主成分被保留，后面的主成分被丢弃。

被丢弃的那些特征值就代表损失掉的方差信息。

本实验用下面的公式计算重构均方误差：

```text
重构 MSE = (原始总方差 - k 维投影方差) / 原始维数
```

也可以写成：

```text
重构 MSE = (lambda_{k+1} + ... + lambda_d) / d
```

所以维度 `k` 越大，保留的特征值越多，丢失的信息越少，重构 MSE 就越小。

## 本实验流程

1. 读取 `ColorHistogram.asc`。
2. 跳过第一列编号，只取后 32 列特征。
3. 对 32 维特征做样本中心化。
4. 计算 32 × 32 协方差矩阵。
5. 对协方差矩阵求特征值。
6. 将特征值从大到小排序。
7. 分别计算降到 1-31 维时的：
   - 投影方差；
   - 重构 MSE；
   - 方差保留率。
8. 把结果写入 CSV 和 JSON 文件。

## 代码块讲解

验收时如果需要讲代码，不需要逐行解释，按下面几个代码块说明即可。

### 1. 路径和输出目录

代码开头定义了：

- 当前脚本路径；
- 数据集路径；
- 输出目录。

作用是让脚本无论从哪里运行，都能找到 `ColorHistogram.asc`，并把结果保存到 `output_py`。

### 2. `load_features()`

这个函数负责读取数据。

重点：

- 数据集第一列是编号；
- 编号不参与 PCA；
- `usecols=range(1, 33)` 表示只读取后 32 维特征。

### 3. `jacobi_eigenvalues()`

这个函数负责求协方差矩阵的特征值。

验收时只需要说明：

- PCA 需要特征值来判断每个主成分的重要程度；
- 这里没有直接调用现成 PCA 或特征分解函数；
- 代码使用 Jacobi 旋转法不断把非对角元素消掉；
- 最后矩阵对角线上的值就是特征值。

不需要展开推导 Jacobi 旋转公式。

### 4. `main()` 中的数据处理

这一部分是 PCA 主流程：

```text
读取数据 -> 中心化 -> 协方差矩阵 -> 特征值 -> 方差和 MSE
```

其中：

- `mean = x.mean(axis=0)`：计算每一维均值；
- `xc = x - mean`：样本中心化；
- `cov = (xc.T @ xc) / n`：计算协方差矩阵；
- `original_variance = sum(eigenvalues)`：计算原始总方差。

### 5. 结果输出

代码最后输出两个文件：

- CSV 文件：适合直接查看 1-31 维的结果表；
- JSON 文件：保存样本数、原始总方差、均值、特征值等汇总信息。

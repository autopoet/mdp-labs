## 推荐验收题目

推荐重点验收：`pca/`。

原因：

1. PCA 的流程更短，更容易在验收时完整讲清楚。
2. 输出指标和题目要求一一对应：原始方差、降维方差、重构 MSE。
3. 32 维降到 1-31 维的结果非常直观，可以直接解释“保留方差越多，重构误差越小”。
4. 相比 VQ，PCA 不涉及码字数量选择、K-means 收敛轮数等可能被追问的工程细节。

## PCA 验收讲解顺序

1. 去掉编号列，只保留 32 维颜色直方图特征。
2. 对每一维求均值，并让每个样本减去均值，完成中心化。
3. 用中心化数据计算协方差矩阵。
4. 对协方差矩阵求特征值，特征值表示对应主成分方向上的方差。
5. 特征值从大到小排序，降到 k 维时取前 k 个主成分。
6. 原始总方差是全部特征值之和。
7. k 维投影方差是前 k 个特征值之和。
8. 重构 MSE 是丢弃特征值之和再除以原始维数。
